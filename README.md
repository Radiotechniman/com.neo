# SZ Neo Z-Wave App for Homey @ Athom.com

Currently supports : 
NAS-WR01ZE - Smart Plug
NAS-DS01Z  - Door Sensor (Yes, can also be used for a window) 

## Flow Options

NAS-WR01ZE 
- You are able to turn switch on/off

NAS-DS01Z
- Add the sensor to the devices page
